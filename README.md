W4 Inbox in AngularJS / REST API
================================

This is a really small and simple sample illustrating how to build an inbox with AngularJS and the
REST API of W4 BPMN+ Engine.


Try it live
-----------

As it's pure HTML/JS you can [try it live](https://rawgit.com/w4software/w4-bpmnplus-angular-inbox-sample/live-demo/index.html)
and enter the credentials to an installation of W4 BPMN+ Engine you have.


Download
--------

You can either download [a zip archive of the source code](https://github.com/w4software/w4-bpmnplus-angular-inbox-sample/archive/master.zip) or clone it using Git.


Build
-----

Run the following

    npm run build

To install the necessary dependencies and wire them in the index file. Then deploy it on an HTTP server to run it.
